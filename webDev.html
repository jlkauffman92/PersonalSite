A website built to learn ruby on rails. Currently hosted via nginx on a Rasberry Pi 3 B. Though currently in desperate need of reCaptcha (coming soon), this website allows users to post annonymously about kind things they have witnessed.
This Very Website
The site was build to learn the basics of Node.js as well as serve as a portfolio. Also, a good deal of CSS was learnt creating this website as every part of this website was built without the use of css frameworks or plugins. This lack of external links makes this site run much faster and makes the source html much simpler. A similar page built with bootstrap or some other framework would have html pages much larger and complicated.
